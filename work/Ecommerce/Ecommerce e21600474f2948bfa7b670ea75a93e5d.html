A clothing eCommerce website that allow users to view apparel products, past orders, car.

✏️Technologies/Skills

The front-end is build with React, React Hooks, React Router, axios and the backend is build with Express, MongoDB, NodeJS.

The checkout payment is handled by Stripe API.

Utilized styled-components to complete the CSS style.
