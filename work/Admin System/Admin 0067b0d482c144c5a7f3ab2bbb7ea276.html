📜Overview

Ecommerce Admin System allows administrators to perform CRUD operations and manage users, products and orders.

✏️Technologies/Skills

The front-end is build with React, React Hooks, React Router, axios and the backend is build with Express, MongoDB, NodeJS.
Compiled and validated test data in Postman and do authentication via JWT.
Integrated Recharts API to setup nice charts and cards to display the number of active users and merchandise sales per month.
Implemented dynamic add/drop form in create page.
Specified naming conventions of local storage and maintain the privacy with CryptoJS.
Solved registration of dynamic routes.
